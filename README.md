# lego-boost-experiment
> node js interface to communicate with LEGO boost module

## Pre-requisites
### Hardware
1. Lego boost module :)

### Development
1. Git
2. Node.js and npm Node ^8.4.0, npm ^5.3.0
3. Workstation with Bluetooth module or external?

## Running the app
1. Go to project root
2. `npm install`
3. `npm run start`
